Rundeck Ruby Client A Ruby wrapper around the Rundeck API.

This library will strive to achieve reliable coverage of the Rundeck API. Please submit an issue if you find a bug and feel free to submit a pull request to contribute fixes or new features.

Initial versions of this gem to not achieve 100% coverage of the Rundeck API. However, there is sufficient support for the primary endpoints that many users will not miss the features at this point. However, if you require support for specific endpoints sooner, please submit an issue indicating which endpoints. I will keep requests in mind when planning future feature support. Each release will include more and more endpoint support until there is 100% coverage.

The layout and the code in this library is inspired by NARKOZ/gitlab. NARKOZ does a fantastic job of keeping code simple and achieving API feature parity.

Note: This guide is geared more toward entry-level users. If you are a more advanced user you may find the Technical Documentation (RDoc) more helpful.

Features

  • Basic CRUD on projects
  • Job endpoints
  • Token endpoints
  • Private/Public Key endpoints
  • Most execution endpoints

Installation

Add this line to your application's Gemfile or gemspec:
# Gemfile
gem 'rundeck', '~> x.x.x'

# gemspec
spec.add_runtime_dependency 'rundeck', '~> x.x.x'
Then execute:
bundle install
Or install it yourself as:
gem install rundeck

Configuration

Require the rundeck gem:
require 'rundeck'
endpointand api_token are required configuration parameters. user_agent is optional and defaults to 'Rundeck Ruby Gem {gem_version}':

# Environment Variables (Recommended!)
export RUNDECK_ENDPOINT=https://rundeck.example.com
export RUNDECK_API_TOKEN=super_secret
# Ruby
Rundeck.endpoint = 'https://rundeck.example.com'
Rundeck.api_token = 'super_secret'
Rundeck.user_agent = 'Custom User Agent'
# Ruby with multiple clients
client1 = Rundeck.client(endpoint: 'https://rundeck.example.com',
                         api_token: 'super_secret',
                         user_agent: 'Custom User Agent')
client2 = Rundeck.client(endpoint: 'https://rundeck2.example.com',
                         api_token: 'super_secret2')
# Ruby on Rails
Rundeck.configure do |config|
  config.endpoint       = 'https://rundeck.example.com'
  config.api_token      = 'super_secret'
  config.user_agent     = 'Custom User Agent'
end
Find your api token by logging in to Rundeck and going to your user dropdown > Profile and selecting the API token
